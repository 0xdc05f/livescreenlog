# LiveScreenLog dashboard

Svelte 5 + Vite UI for browsing and replaying captured sessions.

```bash
npm install
npm run dev      # http://localhost:5173 (proxy API to :8080 if configured)
npm run build    # writes into ../src/main/resources/static
```

Design tokens live in `src/index.css` (warm field-log palette). Product strings are in `src/i18n/`.
